### Blazor Calendar Toy Project
#### 프로젝트 개요

이 프로젝트는 Blazor 학습을 목적으로 진행하는 토이 프로젝트입니다.
기존에 Windows Forms 기반 개발 경험은 있으나, Blazor 및 웹 개발 전반에 대한 이해가 거의 없는 상태에서 달력 컨트롤을 직접 구현하며 Blazor의 핵심 개념을 익히는 것을 목표로 합니다.

---

#### 프로젝트 목표
- Blazor 컴포넌트 기반 UI 개발 흐름 이해
- 상태 기반 렌더링 개념 체득
- WinForms 컨트롤 설계 경험을 Blazor 컴포넌트 설계로 전환
- 단순하지만 구조가 명확한 달력 컨트롤 완성

---

#### 구현 범위
- 지원 기능
  - 단일 날짜 선택
  - 마우스 클릭 입력만 지원
  - 달력 기본 모드
    - 날짜 선택 모드 (월 달력)
    - 월 선택 모드
    - 년도 선택 모드
  - Zoom 기능
    - 날짜 → 월 → 년도 상향 이동
    - 년도 → 월 → 날짜 하향 이동
  - 이전 / 다음 이동 네비게이션

---

#### 기술 스택
- Blazor Server
- C#
- Razor Components
- 기본 CSS (디자인 프레임워크 미사용)

---

#### 저장소 구조
```
src/
 ├─ CalendarToy.App        // Blazor Server 실행 프로젝트
 └─ CalendarToy.Calendar   // 달력 컴포넌트 라이브러리

tests/
 └─ CalendarToy.Calendar.Tests  // 날짜 계산 및 핵심 로직 테스트
```
- 앱과 컨트롤을 분리하여 재사용 가능한 컴포넌트 설계 연습을 목표로 합니다.
- UI와 무관한 날짜 계산 로직은 테스트로 보호합니다.

#### 이 프로젝트의 의미
이 프로젝트는 “완벽한 달력 컨트롤”을 만드는 것이 목적이 아닙니다.
Blazor의 사고방식과 UI 구성 방식을 이해하고, WinForms 개발 경험을 웹 컴포넌트 개발로 확장하는 과정 자체가 목표입니다.
